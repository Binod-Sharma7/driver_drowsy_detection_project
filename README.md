# Driver Drowsiness Detection System

A smart real-time driver safety monitoring system developed using ESP32, IR Eye Blink Sensor, Firebase Realtime Database, and React.js.

## Features
- Real-time drowsiness detection
- IR eye blink sensor monitoring
- Live Firebase dashboard
- Buzzer alert system
- Hazard light activation
- Responsive modern UI
- ESP32 embedded integration

## Technologies Used
- React.js
- Tailwind CSS
- Firebase Realtime Database
- ESP32
- Framer Motion
- Arduino IDE

## System Workflow
1. IR sensor monitors eye blinking activity
2. ESP32 processes sensor data
3. Drowsiness triggers buzzer and hazard light
4. Data updates to Firebase in real time
5. Dashboard displays live monitoring status
